# S.S.S(Stuff.Share.Site)  
## 물건 나눔 사이트      
#### 한국 1인당 년간 플라스틱 쓰레기 배출량은 88KG으로 미.영 다음인 세계 3위 수준이며 서울 / 경기 는 인구수는 900만명/1천300만명으로 총 2천300만명 이라는 통계가 나와 인구수에 비례하여 쓰레기 배출량이 많은것을 확인할 수 있습니다. 따라서 물건 나눔 사이트를 통해 쓰레기 배출량을 줄이고자하는 서비스입니다.
